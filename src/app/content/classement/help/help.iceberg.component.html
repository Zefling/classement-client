Iceberg

Explication

Déplacer les tuiles dans la zone.

Dans les options choisissez le nombre de niveau.

Raccourcis clavier

Ctrl + V
Copier une au plusieurs images qui seront ajouter. Fonctionne aussi avec du texte. Chaque ligne de texte créée une nouvelle tuile.
Ctrl + S
Sauvegarder le brouillon
Ctrl + E
Exporter le brouillon
Clic cental sur une tuile
Remettre dans la liste
Double clic dans une zone
Création d'une tuile à l'endroit visé
